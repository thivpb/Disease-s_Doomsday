# 2.5 Where — Onde?

## Recorte territorial da problemática

> O **Distrito Federal** é o contexto epidemiológico que motiva o jogo: território urbano adensado, com surtos anuais de arboviroses (dengue, zika, chikungunya) e uma rede hospitalar de referência exposta à resistência antimicrobiana. O conteúdo educativo **parte da realidade do DF** e se generaliza para o Brasil. Ver [Why](why.md).

## Onde o jogo é usado e executado

| Contexto                 | Descrição                                                                              |
| ------------------------ | ------------------------------------------------------------------------------------- |
| **Recorte epidemiológico** | Distrito Federal (foco), com leitura aplicável ao Brasil                             |
| **Ambiente de execução** | Desktop local (sem necessidade de internet para jogar)                                |
| **Distribuição**         | GitHub (código-fonte + executável)                                                    |
| **Uso previsto**         | Salas de aula e laboratórios de informática — prioridade para escolas do DF — e uso doméstico |
| **Potencial futuro**     | Servidor web (versão browser via Emscripten/WebAssembly) ou Cloud Gaming              |
| **Parceria-alvo**        | Rede pública de ensino e SES-DF, para uso em campanhas de saúde locais                 |
| **Repositório**          | Felps-26/Disease-s_Doomsday                                                            |
