SPRITES DOS ITENS (drops e coletáveis)
======================================

PNGs dos itens que caem dos inimigos / aparecem no mapa. Sem o arquivo, o item
é desenhado proceduralmente.

Arquivos esperados (nomes EXATOS):
  - mascara.png         -> Máscara Hospitalar (Mundo 1: reduz dano recebido)
  - distanciamento.png  -> Distanciamento Social (Mundo 1: aura que repele)
  - granada_rna.png     -> Desestabilizador de Ácidos Ribonucleicos (granada, ambos os mundos)
  - citocina.png        -> Citocina de Estabilização (regenera vida, ambos os mundos)

Recomendações:
  - Dimensão sugerida: 64 x 64 px.
  - Fundo TRANSPARENTE.
