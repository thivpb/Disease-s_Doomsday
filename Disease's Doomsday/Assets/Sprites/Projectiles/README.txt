SPRITES DOS PROJÉTEIS
=====================

PNGs dos projéteis do jogador e dos inimigos. Sem o arquivo, o projétil é
desenhado proceduralmente.

Arquivos esperados (nomes EXATOS):
  - player_proj.png  -> projétil base do jogador
  - enemy_proj.png   -> projétil base dos inimigos

Recomendações:
  - Dimensão sugerida: 32 x 32 px.
  - Fundo TRANSPARENTE. Apontando para a direita (rotacionado no jogo).
