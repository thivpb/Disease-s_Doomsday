SPRITES DE UI (ícones de HUD / hotbar)
======================================

PNGs de ícones de arma/item para o HUD e a hotbar. Sem o arquivo, os ícones
são desenhados proceduralmente.

Arquivos esperados (nomes EXATOS):
  - icones.png  -> folha (spritesheet) de ícones de armas e itens

Recomendações:
  - Dimensão sugerida: folha de 64 x 64 px por ícone, em grade.
  - Fundo TRANSPARENTE.
  - Quando os ícones individuais forem definidos, a pipeline será estendida com
    o recorte (source rect) de cada um.
