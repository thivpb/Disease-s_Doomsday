SPRITES DAS ARMAS
=================

PNGs das armas equipadas/empunhadas. Sem o arquivo, a arma é desenhada
proceduralmente (weapons_model.c).

Arquivos esperados (nomes EXATOS):
  - espada_seringa.png        -> Espada-Seringa (Mundo 1, melee)
  - rifle_bacteriofagos.png   -> Rifle de Bacteriófagos (Mundo 1, ranged)
  - rifle_vacina.png          -> Rifle de Vacina (Mundo 2, ranged)
  - escalpelizador.png        -> Escalpelizador Estático (Mundo 2, melee anti-escudo)

Recomendações:
  - Dimensão sugerida: 96 x 96 px. Empunhada apontando para a direita
    (a arma é rotacionada no jogo conforme a mira).
  - Fundo TRANSPARENTE.
